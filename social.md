## LinkedIn

🚀 **vex**: an HTTP/3 load testing tool built in Rust.

I built it while developing spooky (a QUIC-based proxy) to better understand HTTP/3 performance characteristics under concurrency.

vex focuses on:
– Controlled concurrency
– Latency percentiles (p50/p90/p95/p99)
– Status code distribution
– Throughput analysis

Both projects are experimental and evolving, but the goal is to explore real-world HTTP/3 behavior from both client and proxy perspectives.

🔗 https://github.com/nishujangra/vex
🔗 https://github.com/nishujangra/spooky
